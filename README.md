# 🦾 AI Personal Trainer - Push-up Tracker
Real-time exercise analysis using **Computer Vision** and **Deep Learning**.

## 🚀 Features
- **Automatic Rep Counting:** Tracks and counts your push-ups accurately.
- **Form Correction:** Detects common mistakes like "Dropped Hips" in real-time.
- **Visual Feedback:** Interactive UI with progress bars and status alerts.

## 🛠️ Tech Stack
- **Python**
- **OpenCV** (Image Processing)
- **MediaPipe** (Pose Estimation)
- **TensorFlow/Keras** (Classification Model)
